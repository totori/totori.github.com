What is Totori?

Totori is an automated User Acceptance testing workbench.

Features

Totori aims to regroup the following features:

  • Behavior-driven testing
  • Automated browser interaction
  • One-click non regression testing of Web applications
  • Test reports with screenshots & video captures

Other Test Automation Tools

There is a bunch of other test automation tools. Here is a non exhaustive list:

  • HP QuickTest Professional

Why is Totori better?

Totori is the first behavior-driven automated browser testing workbench.

It lowers the testing pain and brings quality in the process.
