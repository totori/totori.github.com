h2. About

"Totori":http://totori.org/ is an automated User Acceptance testing workbench that enables behavior-driven testing with automated browser interaction

This Website Template has been designed for its own needs, but is generic enough to be used and reused by anyone else for any other product.

Its style can be modified but should remain visible.
